# Text-to-Image Generation using Stable Diffusion

This project generates images from text prompts using a pre-trained Stable Diffusion model. The model understands the user's description and creates AI-generated images using the diffusion process.

## Features

- Generate images from text prompts
- Uses pre-trained Stable Diffusion model
- User-based prompt input
- GPU accelerated image generation
- Save generated images

## Technologies Used

- Python
- Google Colab
- Stable Diffusion
- Hugging Face Diffusers
- PyTorch

## How It Works

1. User enters a text prompt.
2. Stable Diffusion processes the prompt.
3. The model performs reverse diffusion to generate an image.
4. The final image is displayed and saved.
